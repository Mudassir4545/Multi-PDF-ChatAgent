# 📚 Multi-PDF Chat Agent 🤖

A Streamlit-based AI chatbot that allows users to upload **multiple PDF files** and ask questions about their content using **LangChain, FAISS, and Google's Gemini AI**.

## 🚀 Features
- **Multi-PDF Support**: Upload multiple PDFs and ask questions.
- **FAISS Vector Search**: Efficiently retrieves the most relevant PDF content.
- **Google Gemini AI**: Generates intelligent responses based on retrieved data.
- **Streamlit Interface**: Simple UI for chat-based interactions.
